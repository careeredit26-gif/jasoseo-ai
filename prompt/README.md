## 기업 커버리지 총 307개 + 산업 프레임워크

| 카테고리 | 파일 | 기업 수 |
|----------|------|---------|
| 대기업 그룹(삼성/SK/LG/현대/CJ/한화/포스코) | 03 | 37 |
| 대기업 확장(롯데/GS/두산/신세계/LS/효성/HD현대) + 주요 업종별 | 03b | 64 |
| 금융(은행/증권/보험/카드/자산운용/공적금융) | 03d | 52 |
| IT/게임/콘텐츠/스타트업 | 03e | 38 |
| 제조/화학/건설/방산/에너지/부품 | 03f | 48 |
| 유통/식품/뷰티/물류/항공/서비스 | 03g | 40 |
| 공기업/공공기관 | 03h | 28 |
| **소계(직접 등록)** | | **307** |
| 산업 프레임워크(15개 산업 + 외국계) | 03c | ∞ |

## 기업 탐색 우선순위
```
사용자가 기업명 입력
  ↓
[1] 03 → 03b → 03d~03h에서 검색 (307개 직접 매칭)
  ↓ 없으면
[2] 03c 산업 프레임워크에서 산업군 추론
  ↓ 동시에
[3] 사용자에게 기업 인재상/채용 페이지 정보 요청
```

## 사용 방법

### GPT/Claude 커스텀 프롬프트 적용
- **전체 투입**: 12개 파일 모두 → 가장 정확
- **선택 투입**: 01 + 02 + 해당 기업이 있는 DB파일 + 04 + 05 + 07
- **최소 투입**: 01 + 03c(산업 프레임워크)만으로도 모든 기업 기본 대응 가능
